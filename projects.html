CMD scrips for automation and smoother workflow
These are the .bat and possibly other commandline/shell scripts, that I have made, to simplify my day-to-day workflow. They include both aliases for renaming cmd commands to be either easier to remember or more similar to the Linux / Unix counterparts, as well as shortcuts to open applications from commandline.
This also includes my "lazy git" script, for committing and pushing git repos from the commandline more easily and faster, taking also care of navigating to the repository root folder.
Light picture downloader

Genshin Impact ingame satistics viewer
A small project written to make it easier to see ingame state without starting the game, and automate the "outside the game" reward mechanics: getting the daily login rewards and searching + redeeming giftcodes for ingame rewards. An exercise in using and reverse engineering api-s.
